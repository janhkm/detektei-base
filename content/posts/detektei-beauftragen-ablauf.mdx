**So beauftragen Sie eine Detektei – der komplette Ablauf in 5 Schritten:**

1. **Erstgespräch:** Unverbindliche Beratung, Fall schildern, Machbarkeit prüfen
2. **Vertrag:** Schriftliche Vereinbarung mit klarem Kostenlimit
3. **Ermittlung:** Diskrete Durchführung durch erfahrene Ermittler
4. **Updates:** Regelmäßige Statusberichte während der Observation
5. **Abschlussbericht:** Gerichtsverwertbare Dokumentation aller Ergebnisse

Sie haben sich entschieden, eine Detektei einzuschalten – aber wie geht es jetzt weiter? Der Gedanke, einen Privatdetektiv zu beauftragen, kann befremdlich wirken. Doch der Ablauf ist professioneller und strukturierter, als viele denken.

In diesem Ratgeber führen wir Sie Schritt für Schritt durch den gesamten Prozess: vom ersten Kontakt über die eigentliche Ermittlung bis zum Abschlussbericht. So wissen Sie genau, was Sie erwartet, wenn Sie eine Detektei engagieren.

<KeyTakeaways>
- **Jetzt anrufen:** Unverbindliches Gespräch zur Einschätzung Ihres Falls
- **Schriftlicher Vertrag:** Pflicht vor Beginn jeder Ermittlung
- **Kostenlimit vereinbaren:** Sie behalten die Kontrolle über die Ausgaben
- **Regelmäßige Updates:** Sie werden über den Fortschritt informiert
- **Gerichtsverwertbarer Bericht:** Professionelle Dokumentation am Ende
- **Jederzeit abbrechen:** Sie können den Auftrag beenden
</KeyTakeaways>

## Detektei beauftragen: Voraussetzungen & Vorbereitung

**Kurz:** Um eine Detektei erfolgreich zu beauftragen, benötigen Sie einen konkreten Verdacht, grundlegende Informationen zur Zielperson und ein berechtigtes Interesse. Je besser Sie vorbereitet sind, desto effizienter kann der Detektiv Auftrag ablaufen.

### Was Sie vorbereiten sollten

Bevor Sie eine Detektei kontaktieren, sammeln Sie folgende Informationen. Eine ausführliche Anleitung finden Sie unter [Welche Infos braucht ein Detektiv von mir?](/blog/informationen-fuer-detektiv).

| Information | Warum wichtig? |
|-------------|----------------|
| Name der Zielperson | Grundlegende Identifikation |
| Beschreibung (Aussehen, Alter) | Für Observationen |
| Bekannte Adressen | Wohnung, Arbeit, Treffpunkte |
| Fahrzeug (wenn bekannt) | Kennzeichen, Marke, Farbe |
| Gewohnheiten und Routinen | Arbeitszeiten, Aktivitäten |
| Zeitfenster | Wann ist Observation sinnvoll? |
| Bisherige Erkenntnisse | Was wissen Sie bereits? |
| Ihr konkretes Ziel | Was genau wollen Sie beweisen? |

### Was Sie NICHT tun sollten

- **Selbst ermitteln:** Gefahr der Entdeckung und rechtliche Risiken – mehr dazu unter [Detektei oder selbst ermitteln?](/blog/detektei-oder-selbst-ermitteln)
- **Die Zielperson konfrontieren:** Verhalten wird angepasst
- **Informationen erfinden:** Nur Tatsachen, keine Vermutungen

## Schritt 1: Das Erstgespräch – Ihr Fall wird geprüft

**Kurz:** Im Erstgespräch schildern Sie Ihre Situation, der Detektiv prüft die Machbarkeit und Sie erhalten eine erste Einschätzung zu den Erfolgschancen. Jetzt anrufen – unverbindlich.

### Wie das Erstgespräch abläuft

**Kontaktaufnahme:** Per Telefon, E-Mail oder Kontaktformular. Nennen Sie kurz Ihr Anliegen – Details folgen im Gespräch. Sie können auch [online eine Detektei beauftragen](/blog/detektei-online-beauftragen) oder [anonym anfragen](/blog/detektei-anonym-anfragen), wenn Ihnen Diskretion besonders wichtig ist.

**Termin:** Telefonisch, per Video oder persönlich im Büro der Detektei. Wählen Sie, was für Sie am bequemsten ist.

**Inhalt des Gesprächs:**

1. Sie schildern Ihre Situation ausführlich
2. Der Detektiv stellt Rückfragen
3. Rechtliche Machbarkeit wird geprüft
4. Erste Einschätzung zu Vorgehen und Kosten
5. Offene Fragen werden beantwortet

### Was Sie im Erstgespräch fragen sollten

- Wie schätzen Sie die Erfolgschancen ein?
- Welche Methoden würden Sie einsetzen?
- Was kostet die Ermittlung voraussichtlich?
- Wie lange wird es dauern?
- Was passiert, wenn nichts herauskommt?
- Wie werden die Beweise dokumentiert?

### Warnsignale im Erstgespräch

Ein seriöser Detektiv wird:

- **Ehrlich** auch negative Einschätzungen geben
- **Grenzen** aufzeigen (was nicht möglich ist)
- **Nicht drängen** auf sofortige Entscheidung
- **Keine Erfolgsgarantien** versprechen

Mehr dazu: [Seriöse Detektei erkennen](/blog/serioese-detektei-erkennen)

Ausführliche Tipps zur Vorbereitung finden Sie unter [Erstgespräch mit einer Detektei – Vorbereitung](/blog/erstgespraech-detektei).

## Schritt 2: Die Auftragserteilung – Vertrag und Konditionen

**Kurz:** Nach der Entscheidung, einen Privatdetektiv zu beauftragen, wird ein schriftlicher Vertrag abgeschlossen. Er regelt Umfang, Kosten, Datenschutz und Ihre Rechte. Erst nach Vertragsabschluss beginnt die Ermittlung.

### Was im Vertrag geregelt wird

| Vertragsbestandteil | Bedeutung für Sie |
|--------------------|-------------------|
| **Auftragsumfang** | Was genau soll ermittelt werden? |
| **Methoden** | Welche Ermittlungsarten sind geplant? |
| **Zeitrahmen** | Von wann bis wann läuft der Auftrag? |
| **Kostenrahmen** | Maximum, Stundensatz, Nebenkosten |
| **Kostenlimit** | Obergrenze, ab der Sie kontaktiert werden |
| **Datenschutz** | Wie werden Ihre Daten geschützt? |
| **Berichterstattung** | Wann und wie werden Sie informiert? |
| **Kündigung** | Wie können Sie den Auftrag beenden? |

### Mit welchen Kosten müssen Sie rechnen?

Die Kosten für eine Detektei werden individuell nach Fall und Aufwand berechnet. Faktoren wie Art der Ermittlung, Dauer und Komplexität bestimmen den Preis. Im Erstgespräch erhalten Sie eine transparente Einschätzung für Ihre Situation.

Detaillierte Informationen finden Sie auf unserer [Kosten-Seite](/kosten).

### Das Kostenlimit – Ihre Sicherheit

Eine wichtige Vereinbarung: das Kostenlimit. Sie legen einen Betrag fest, bei dessen Erreichen der Detektiv Sie kontaktiert. So vermeiden Sie böse Überraschungen.

**Beispiel:** Sie vereinbaren ein Kostenlimit von 2.000€. Bei Erreichen dieses Betrags informiert Sie der Detektiv und Sie entscheiden gemeinsam über das weitere Vorgehen.

### Die Anzahlung

Branchenüblich ist eine Anzahlung (Akonto) vor Beginn der Ermittlung. Sie beträgt typischerweise 30-50% des geschätzten Gesamtauftrags und wird mit der Endabrechnung verrechnet.

**Wichtig:** Ohne schriftlichen Vertrag sollten Sie keine Anzahlung leisten.

### Praxisbeispiel: Vom Erstgespräch zur erfolgreichen Ermittlung

*Anonymisierter Fall einer Partnerdetektei – Veröffentlichung vom Mandanten genehmigt:*

Ein Mandant aus dem Rhein-Main-Gebiet hatte den Verdacht auf Untreue. Im Erstgespräch schilderte er konkrete Anzeichen: veränderte Routinen, häufige "Überstunden" am Wochenende, ein neues Passwort am Handy.

**Der Ablauf im Überblick:**
- **Montag:** Erstgespräch (45 Minuten, kostenlos)
- **Dienstag:** Vertragsabschluss mit vereinbartem Kostenlimit
- **Freitag-Sonntag:** Erste Observation (Wochenende 1)
- **Samstag (Wochenende 2):** Eindeutige Dokumentation – Zielperson trifft sich mit dritter Person
- **Montag:** Abschlussbesprechung mit Berichtübergabe

**Ergebnis:** Nach nur 4 Observationstagen lagen gerichtsverwertbare Beweise vor. Der Mandant hatte Klarheit – und die Grundlage für sein weiteres Vorgehen.

**Lehre:** Eine gute Vorbereitung im Erstgespräch (konkrete Zeiten, Verdachtsmomente) ermöglichte eine effiziente und damit kostengünstigere Ermittlung.

## Schritt 3: Die Durchführung – Ermittlung läuft

**Kurz:** Nach Vertragsabschluss beginnt die eigentliche Ermittlung. Der Detektiv arbeitet diskret und professionell. Sie werden nicht in die operative Durchführung einbezogen, erhalten aber regelmäßige Updates.

### Was während der Ermittlung passiert

Je nach Auftragsart führt der Detektiv durch:

**Bei Observationen:**
- Diskrete Beobachtung der Zielperson
- Fotodokumentation im öffentlichen Raum
- Protokollierung aller Aktivitäten
- Dokumentation von Kontaktpersonen

Die meisten Observationen dauern [3-4 Tage](/blog/observation-dauer) – selten länger. Was ein Detektiv dabei darf und was nicht, erfahren Sie unter [Was darf ein Detektiv?](/blog/was-darf-ein-detektiv).

**Bei Personensuchen:**
- Recherche in öffentlichen Datenbanken
- OSINT-Recherche (Open Source Intelligence)
- Befragung von Personen (freiwillig)
- Ortung über verschiedene Methoden

**Bei Wirtschaftsermittlungen:**
- Überwachung verdächtiger Mitarbeiter
- Dokumentation von Arbeitszeiten/Verhalten
- Recherche zu Nebentätigkeiten
- Beweissicherung für Arbeitsrecht

Für Unternehmen gelten besondere Anforderungen – alle Details finden Sie unter [Wirtschaftsdetektei](/wirtschaftsdetektei).

### Was Sie während der Ermittlung tun sollten

- **Erreichbar bleiben:** Für Rückfragen und Updates
- **Diskret bleiben:** Sprechen Sie nicht über die Ermittlung
- **Normal verhalten:** Ändern Sie nichts an Ihrer Routine
- **Nicht selbst eingreifen:** Überlassen Sie die Arbeit dem Profi

### Was Sie NICHT tun sollten

- Die Zielperson konfrontieren
- Selbst "mitermitteln"
- Dritten von der Beauftragung erzählen
- Den Detektiv ständig kontaktieren (Updates kommen)

## Schritt 4: Status-Updates – Sie bleiben informiert

**Kurz:** Seriöse Detekteien halten Sie mit regelmäßigen Updates auf dem Laufenden. Die Häufigkeit hängt vom Auftrag ab – bei wichtigen Entwicklungen werden Sie sofort informiert.

### Wie die Kommunikation läuft

| Situation | Kommunikation |
|-----------|---------------|
| Routinemäßiger Fortschritt | Regelmäßige Updates (z.B. täglich bei Observation) |
| Wichtige Entwicklung | Sofortige Benachrichtigung |
| Kostenlimit erreicht | Kontakt zur Abstimmung |
| Keine Ergebnisse | Ehrliche Information |
| Abschluss | Terminvereinbarung für Besprechung |

### Was Sie aus den Updates erfahren

- Aktueller Stand der Ermittlung
- Relevante Beobachtungen
- Aufgelaufene Kosten
- Geschätzter weiterer Zeitbedarf
- Empfehlungen für das Vorgehen

### Ihre Steuerungsmöglichkeiten

Sie behalten die Kontrolle:

- **Auftrag erweitern:** Wenn neue Erkenntnisse andere Fragen aufwerfen
- **Fokus ändern:** Wenn sich Prioritäten verschieben
- **Auftrag beenden:** Jederzeit möglich
- **Budget anpassen:** Kostenlimit erhöhen oder senken

## Schritt 5: Der Abschlussbericht – Ihre Ergebnisse

**Kurz:** Am Ende der Ermittlung erhalten Sie einen detaillierten, gerichtsverwertbaren Bericht. Er dokumentiert alle Beobachtungen, enthält Beweismaterial und gibt Empfehlungen für das weitere Vorgehen.

### Was der Bericht enthält

| Element | Inhalt |
|---------|--------|
| **Deckblatt** | Auftraggeber, Zeitraum, Zusammenfassung |
| **Chronologie** | Alle Beobachtungen mit Datum und Uhrzeit |
| **Fotodokumentation** | Nummerierte Bilder mit Zeitstempel |
| **Zusammenfassung** | Die wichtigsten Erkenntnisse |
| **Bewertung** | Objektive Einschätzung (ohne Spekulation) |
| **Empfehlungen** | Vorschläge für das weitere Vorgehen |
| **Unterschrift** | Bestätigung durch den Detektiv |

### Die Abschlussbesprechung

Nach Fertigstellung des Berichts vereinbaren Sie einen Termin zur Besprechung:

1. **Bericht übergeben:** Sie erhalten die Dokumentation
2. **Ergebnisse erläutern:** Der Detektiv erklärt die Erkenntnisse
3. **Fragen klären:** Ihre offenen Fragen werden beantwortet
4. **Nächste Schritte:** Was können Sie mit den Ergebnissen tun?
5. **Abrechnung:** Endabrechnung wird erstellt

### Was Sie mit dem Bericht tun können

- **Rechtsanwalt:** Beweise in Verfahren einbringen
- **Arbeitgeber:** Bei Wirtschaftsermittlungen für HR-Maßnahmen
- **Persönlich:** Grundlage für Gespräche oder Entscheidungen
- **Gericht:** Detektiv kann als Zeuge aussagen

## Detektei beauftragen: Häufige Probleme und Lösungen

**Kurz:** Nicht jede Ermittlung verläuft wie geplant. Wenn Sie eine Detektei beauftragen, sollten Sie wissen, wie Sie mit typischen Herausforderungen umgehen.

### Problem: Keine Ergebnisse

Manchmal liefert eine Ermittlung kein eindeutiges Ergebnis. Das kann bedeuten:

- Der Verdacht hat sich nicht bestätigt
- Die Zielperson verhält sich vorsichtig
- Das Zeitfenster war nicht optimal

**Lösung:** Ein ehrlicher Detektiv wird das offen kommunizieren und mit Ihnen besprechen, ob eine Fortsetzung sinnvoll ist.

### Problem: Kosten übersteigen Budget

Manchmal dauert eine Ermittlung länger als geplant.

**Lösung:** Das Kostenlimit schützt Sie. Bei Erreichen werden Sie kontaktiert und entscheiden gemeinsam.

### Problem: Zielperson wird misstrauisch

Selten, aber möglich: Die Zielperson bemerkt etwas.

**Lösung:** Professionelle Detektive erkennen das früh und passen ihr Vorgehen an oder unterbrechen die Ermittlung.

## Checkliste: Detektei beauftragen – optimale Vorbereitung

**Kurz:** Mit dieser Checkliste sind Sie optimal vorbereitet, wenn Sie eine Detektei beauftragen möchten.

### Vor dem Erstgespräch

- [ ] Informationen zur Zielperson gesammelt
- [ ] Eigenes Ziel klar definiert
- [ ] Zeitfenster für mögliche Observation überlegt
- [ ] Budget-Rahmen festgelegt
- [ ] Fragen an die Detektei notiert

### Vor der Beauftragung

- [ ] Mehrere Detekteien verglichen
- [ ] Seriosität geprüft (Checkliste verwenden)
- [ ] Vertrag gelesen und verstanden
- [ ] Kostenlimit vereinbart
- [ ] Kommunikationsweg festgelegt

### Während der Ermittlung

- [ ] Erreichbar bleiben
- [ ] Diskret bleiben
- [ ] Nicht selbst eingreifen
- [ ] Updates abwarten

## Fazit: Detektei beauftragen leicht gemacht

Der Ablauf, wenn Sie eine Detektei beauftragen, ist strukturierter und professioneller, als viele erwarten. Mit der richtigen Vorbereitung und einem seriösen Detektivbüro verläuft der Prozess reibungslos – von der ersten Anfrage bis zum Abschlussbericht.

Das Wichtigste: Nehmen Sie sich Zeit für die Auswahl, wenn Sie einen Privatermittler engagieren möchten. Nutzen Sie das Erstgespräch und klären Sie alle Fragen, bevor Sie einen Vertrag unterschreiben. Ein seriöser Ermittler wird Sie professionell durch den gesamten Prozess begleiten.

**Eilig?** In dringenden Fällen kann eine Detektei auch [sofort starten](/blog/detektiv-sofort-beauftragen). Informieren Sie sich auch über unseren [standardisierten Ablauf](/ablauf), die [transparente Kostenstruktur](/kosten) und unsere [Einsatzgebiete deutschlandweit](/einsatzgebiete).

**Bereit für den ersten Schritt?** Rufen Sie jetzt an – unverbindlich. Wir hören uns Ihren Fall an und geben Ihnen eine ehrliche Einschätzung.
